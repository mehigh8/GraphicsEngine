# GraphicsEngine
<b>Roșu Mihai Cosmin</b>

Project made for the Real-Time 3D Graphics Engines course of the Graphics, Multimedia and Virtual Reality master program.

## Details

A fully functional 3D graphics engine made in OpenGL having the following features:
- 3D objects importer
- Texture support
- FPS Camera imeplementation
- Keyboard/Mouse events
- Lighting support
- XML scene saving/loading
- UI Elements

A detailed documentation can be found at: [Documentation.pdf](https://github.com/mehigh8/GraphicsEngine/blob/main/Documentation.pdf)
